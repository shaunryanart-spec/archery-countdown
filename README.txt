ARCHERY COUNTDOWN APP

How to run:
1. Publish the whole folder to Netlify, GitHub Pages, or another HTTPS host.
2. Open the public site once while online.
3. After that, it will continue to work offline from the same browser/device.
4. On supported phones, use “Add to Home Screen” to install it like an app.

Features:
- Selectable 2, 3, or 4 minute shooting time.
- 10-second yellow “Archers to the line” countdown.
- 2 buzzes to call archers to the line.
- 1 buzz to begin shooting.
- Green screen until the final 30 seconds.
- Yellow screen for the final 30 seconds.
- 3 buzzes and red screen at zero.
- Score button stops and resets at any time.
- Fullscreen and screen-wake support.

Keyboard:
- Space: Start
- S: Score/reset
- F: Fullscreen

Important:
Offline caching works when the app is served over HTTPS or localhost. It does not work reliably when index.html is opened directly as a local file.
